# Employee Management System (Django)

This is a web-based Employee Management System developed using Django. The application allows users to manage employee records efficiently through CRUD operations.

## Features
- Add new employees
- View employee details
- Update employee information
- Delete employee records

## Technologies Used
- Python
- Django
- HTML5, CSS3
- SQLite (default database)

## Concepts Used
- CRUD Operations (Create, Read, Update, Delete)
- Django ORM (Object Relational Mapping)
- MVT Architecture (Models, Views, Templates)

## Project Structure
- ems_project/ → Django project settings
- employee/ → Application module
- templates/ → HTML templates
- manage.py → Project management file
